    Cache Simulator

  • C++

    • Developed C++ code to simulate different types of cache memories ( Direct mapped, Fully Associative and Set associative). The cache used Write Back Write Allocate (WBWA) policy and Least Recently Used (LRU) replacement policy. The simulator was designed to implement prefetching to study reduction in cache miss rate. Different parameters were varied to obtain values of Average Access Time (AAT), Miss rate etc. The simulator was evaluated using different SPEC benchmarks and a simulator best suited for each benchmark was found.

    Soil Moisture Detector
